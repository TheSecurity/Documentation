### Weather

{% hint style="success" %}
@Manager\#9545 **weather \<\town\>**
{% endhint %}

{% hint style="info" %}
Sends weather forecast about a selected town
{% endhint %}

```text
@Manager#9545 weather Prague
```

### Random Picture

@Manager\#9545 **randomPicture \<\width\> \<\height\> \<\params\>**
{% endhint %}

{% hint style="info" %}

{% endhint %}

```text
@Manager#9545 400 300 params
```
